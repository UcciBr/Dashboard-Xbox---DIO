# Dashboard-Xbox---DIO

# O objetivo deste desafio é criar um dashboard de vendas, com foco na organização e visualização de dados. O objetivo é transformar dados brutos em informações visuais claras e úteis, permitindo uma análise eficaz do desempenho de vendas e a tomada de decisões baseadas em dados.

# Utilizando uma base de dados disponibilizada em .xlsx, construi um dashboard interativo possbilitando a resposta para 3 questionamentos de alta gerência:
  1. Qual o faturamento total de vendas de planos anuais (contendo todas as assinaturas agregadas);
  2. Qual o faturamento total de vendas dos planos anuais, separado por auto renovação (yes) e auto renovação (no);
  3. Total de vendas de assinaturas do EA Play;
  4. Total de vendas de assinaturas do Minecraft.

# É possivel filtrar por valores Mensais - Trimestrais - Anuais.
